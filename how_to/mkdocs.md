# How to deploy the documentation

# Required packages:
1. mkdocstrings
2. mkdocs-material
3. mkdocstrings-python

# Required files:
1. mkdocs.yml

# Usage:
1. To build the documentation locally:
`mkdocs serve`

2. To publsh the documentation:
First, commit and push all changes to the repository.
Then, run: `mkdocs gh-deploy`
